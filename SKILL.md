# Agent Builder — AI Agent构建助手

## 功能说明

帮助设计和构建AI Agent系统。

## 使用方法

### 1. Agent架构设计

```
用户: 设计一个客服Agent，需要具备什么能力？
```

输出：
- 核心能力定义
- 知识库配置
- 工具集设计
- 对话策略

### 2. 系统提示词编写

```
用户: 为数据分析Agent编写系统提示词
```

### 3. 工具配置

```
用户: 这个Agent需要接入哪些工具？
```

### 4. Agent评估

```
用户: 评估这个Agent设计是否合理
```

## 示例输出

```
Agent设计：数据分析助手

【核心定位】
- 角色：专业数据分析师
- 目标：帮助用户理解数据、发现洞察

【能力矩阵】
| 能力 | 优先级 | 说明 |
|------|--------|------|
| 数据分析 | P0 | 核心能力 |
| 可视化 | P1 | 重要能力 |
| 预测分析 | P2 | 扩展能力 |

【工具集】
- 数据读取 (CSV/Excel/JSON)
- 统计分析
- 图表生成
- 报告导出

【系统提示词】
你是一个专业的数据分析师...
```
